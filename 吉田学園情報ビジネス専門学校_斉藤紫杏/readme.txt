◆01_Zombie Survivor
個人制作です。
銃やギミックでゾンビを倒し、スコアを稼ぐゲームになっています。
最終的にはボスを倒したらゲームクリアになっています。

アピールポイント
今回作成したゲームのアピールしたいところは、カメラの移動や敵を倒す方法です。
カメラの移動に関しては、コントローラーのスティックで視点移動が自由にできて向きたいところを向くことができます。
そしてエイムをしたときに自然な感じにエイム画面に行くことができるのがとてもこだわった点です。
敵を倒す方法については、最初から所持している銃で倒すことができ、マップ内に複数箇所あるギミックでも倒すことができます。
